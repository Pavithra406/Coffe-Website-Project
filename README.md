☕ Coffee Website

A modern and responsive Coffee Shop Website built using HTML, CSS, and JavaScript.
This project focuses on creating an attractive user interface for a coffee shop, showcasing different coffee products, services, and a visually engaging layout.

The website demonstrates the use of animations, responsive design, and interactive elements to enhance the user experience.

🚀 Project Objective

The main objective of this project is to:

Practice frontend web development
Build a responsive website layout
Implement animations and interactive UI components
Learn how to integrate JavaScript libraries in web projects
Design a visually appealing product showcase website
✨ Features

☕ Modern coffee shop themed design
📱 Responsive layout for different screen sizes
🎞️ Scroll-based animations using ScrollReveal
🔄 Interactive sliders using Swiper.js
🎨 Clean and attractive user interface
⚡ Smooth user experience with JavaScript effects

🔹 html.html

The main HTML file that defines the structure of the coffee website.

🔹 style.css

Contains all the styling and layout design for the website.

🔹 script.js

Handles the interactive behavior of the website such as animations and dynamic elements.

🔹 scrollreveal.min.js

A JavaScript library used to create scroll-based animations.

🔹 swiper-bundle.min.js & swiper-bundle.min.css

Used to create interactive sliders and carousels for displaying content.
